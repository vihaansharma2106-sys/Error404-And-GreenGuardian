# Error404-And-GreenGuardian

This project is a plant health monitor and it focuses on the problems like whether it needs more water, more light, less temrature ,if there is sufficient humidity and soil moisture and it tracks these conditions. We have used components like DHT11, TFT LCD 1.8, Soil moisture sensor, a photoresistor, a water level detector. This project is made for everyday use for people who cannot give enough time to take care of their plants. It combines robotics and biology to become an everyday smart gadget, which can be very useful to people with plants
